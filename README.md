# 14-DOF 四轮独立驱动/转向 底盘动力学模型

## 模型概述

本模型在参照论文《考虑坡度影响的车辆行驶动力学建模与仿真》(邓涛 等, 2012) 的基础上，扩展为 **四轮独立驱动 (4WID) + 四轮独立转向 (4WIS)** 的14自由度底盘动力学模型。

## 自由度分布 (14 DOF)

| 序号 | 自由度 | 符号 | 说明 |
|------|--------|------|------|
| 1 | 纵向 | \( v_x \) | 车身纵向速度 |
| 2 | 横向 | \( v_y \) | 车身横向速度 |
| 3 | 垂向 | \( z \) | 簧上质量垂向位移 |
| 4 | 侧倾 | \( \phi \) | 车身侧倾角 |
| 5 | 俯仰 | \( \theta \) | 车身俯仰角 |
| 6 | 横摆 | \( \dot{\psi} \) | 横摆角速度 |
| 7-10 | 车轮垂向 | \( z_{w,1:4} \) | 四轮垂向位移 |
| 11-14 | 车轮旋转 | \( \omega_{w,1:4} \) | 四轮旋转角速度 |

## 状态向量 (29 维)

```
x = [vx, vy, z, vz, φ, φ_dot, θ, θ_dot, ψ_dot,
     z_w(1:4), z_w_dot(1:4), ω_w(1:4),
     Fx_relax(1:4), Fy_relax(1:4)]
```

含8个轮胎松弛力状态，用于模拟轮胎力的动态滞后效应。

## 控制输入 (11 维)

| 输入 | 符号 | 说明 |
|------|------|------|
| 转向角 ×4 | \( \delta_{1:4} \) | 四轮独立转向 |
| 电机扭矩 ×4 | \( T_{m,1:4} \) | 四轮独立驱动 |
| 制动踏板 | \( \beta_{brk} \) | 制动开度 [0~1] |
| 坡度比 ×2 | \( i_h, i_c \) | 横向/纵向坡度 |

## 关键建模特征

### 四轮独立驱动
- 四个轮毂/轮边电机，无中央传动链
- 电机模型含恒转矩区 + 恒功率区
- 支持扭矩矢量控制 (TVC) 和直接横摆力矩控制 (DYC)

### 四轮独立转向
- 各轮独立转向角输入
- 支持蟹行、原地转向等高级机动模式
- 运动学方程含各轮独立坐标变换

### 非线性悬架 (模型D)
```
F_si = F_s0i + K_s1_i × stroke + K_s2_i × stroke³
     + C_s1_i × stroke_dot + C_s2_i × |stroke_dot| × stroke_dot
```
- 三次弹簧项模拟缓冲块非线性刚度
- 平方阻尼项模拟压缩/回弹阻尼不对称

### 轮胎模型
- Magic Formula 纯侧偏/纯纵滑
- 摩擦椭圆联合工况修正
- 一阶松弛长度动力学

### 坡度建模
- 横向 + 纵向坡度独立输入
- 坡度引起的三轴重力分量
- 坡度载荷转移（纵向轴荷 + 横向轮荷转移）

## 使用方法

### MATLAB 环境
```matlab
>> run_sim
```

自动执行:
1. 加载车辆参数
2. 6种工况 ODE 仿真（含蟹行、原地转向等）
3. 可视化输出 (7类曲线)
4. Simulink 模型构建

### 自定义仿真
```matlab
p = vehicle_params();

% 定义控制输入
% u = [delta(1:4), T_m(1:4), beta_brk, i_h, i_c]
u = [deg2rad([3, 3, 1, 1]), ...  % 四轮转向角
     100, 100, 60, 60, ...       % 四轮电机扭矩
     0, ...                      % 制动
     0, 0.1];                    % 坡度

% 运行仿真
R = simulate(p, u, [], 10);

% 画图
plot_results(R);
```

## 与旧模型的主要差异

| 特征 | 旧14DOF模型 | 新14DOF模型 |
|------|------------|------------|
| 驱动形式 | 发动机→CVT→差速器 | 四轮独立电机 |
| 转向 | 仅前轮转向 | 四轮独立转向 |
| 悬架 | 线性弹簧阻尼 | 非线性模型D |
| 控制输入 | 6路 | 11路 |
| 适用场景 | 常规坡度工况 | 坡度 + 四轮转向 + 扭矩矢量 |
